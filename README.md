## وصف المشروع
مشروع API لإدارة المنتجات والكارت والطلبات باستخدام Laravel 11 وSanctum.  
يدعم التعامل مع المنتجات متعددة اللغات (عربي وإنجليزي) باستخدام Spatie Translatable.

## المتطلبات
- PHP >= 8.1
- Composer
- MySQL
- Laravel 11
- Node.js (اختياري لو عندك واجهة)
